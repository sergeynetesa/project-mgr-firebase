Project Management Application
Project Management Application – example of Angular client Single Page Application (SPA) with Firebase Authentication and Cloud Firestore NOSQL database.
The Application was developed using
  • Angular 9 platform and framework
  • Angular Material 9 - Material Design components for Angular
  • Angular CLI
  • Firebase Authentication
  • Cloud Firestore NOSQL database
  • HTML 5 and CSS 3, Style preprocessors:  SCSS, SASS, LESS
  • TypeScript and ECMAScript 2015 (ES6 JavaScript)
  • RxJS 6 (Reactive Extensions Library for JavaScript)

The Firebase Authentication provides backend services, SDKs to authenticate Users to the Application.
It supports Authentication using Email/Passwords provider. The Application Backend – Cloud Firestore NOSQL database.

Used Angular buildings blocks:
  • NgModules, Components, Directives, Pipes
  • Services and Dependency Injection (DI)
  • Reactive Forms and Forms Validators
  • RxJS Observables and Subjects
  • Router and Guards
The Application functions:
  • signup (create) User account
  • login User by user Email and Password
  • create, read, update, delete (CRUD operations) on one or many Projects
  • in selected Project create, read, update, delete (CRUD operations) on one or many Tasks

Data model:
User (1 : M) Projects (1 : M) Tasks
